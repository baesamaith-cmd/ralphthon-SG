# Search Fixtures

These vague-memory queries are used to judge whether LinkTrace search works from human memory rather than exact titles.

| Query | Intended top result |
| --- | --- |
| `agent memory` | Codex agent workflow notes or Ouroboros spec loop repo |
| `watch later` | YouTube walkthrough on AI note taking |
| `sleep timing` | Sleep hygiene checklist |
| `career growth` | Engineering career essay |
| `login gated` | Social post with limited metadata |

The search implementation matches title, summary, recall cues, tags, and domain. Recall cues carry the highest weight because they are the product's main memory affordance.
